# PMatrix 🖥️⚡

**PMatrix** — это легковесный и быстрый клон легендарной консольной утилиты `cmatrix`, написанный на Python специально для операционных систем Windows. Программа воссоздает знаменитый эффект падающих зеленых символов (цифровой дождь) из фильма «Матрица» прямо в вашем терминале.

В отличие от оригинального CMatrix, который жестко привязан к Unix-системам, этот проект работает на Windows «из коробки» благодаря библиотеке `windows-curses` и поддерживает кастомные аргументы запуска.

## 🔥 Особенности (Features)

* 🚀 **Полная поддержка Windows:** работает в CMD, PowerShell и встроенных терминалах (например, в VS Code).
* 🎨 **Кастомизация цветов:** выбор из 8 доступных цветов терминала через аргументы командной строки.
* 🕶️ **Эффект свечения:** случайные символы в «голове» падающего потока подсвечиваются ярким белым цветом и жирным шрифтом.
* 📏 **Автоматический ресайз:** анимация корректно подстраивается под изменение размеров окна терминала на лету.
* 📦 **Автономность:** скрипт можно упаковать в один `.exe` файл, который будет запускаться на любом ПК без установленного Python.

## 🛠️ Инструкция по запуску (Usage)

### 1. Установка зависимостей
Для работы скрипта требуется порт библиотеки curses для Windows:
```bash
pip install windows-curses
```

### 2. Обычный запуск (Зеленая матрица)
```bash
python matrix.py
```

### 3. Запуск с выбором цвета
Используйте флаг `-color` (или `--color`), чтобы задать нужный оттенок:
```bash
python matrix.py -color red
```
*Доступные цвета:* `green` (по умолчанию), `red`, `blue`, `cyan`, `yellow`, `magenta`, `white`, `black`.

### 4. Просмотр справки
```bash
python matrix.py -h
```

## 📦 Сборка в самостоятельный `.exe` файл

Если вы хотите превратить скрипт в полноценное Windows-приложение, используйте `PyInstaller`:

1. Установите упаковщик:
   ```bash
   pip install pyinstaller
   ```
2. Соберите проект одной командой:
   ```bash
   pyinstaller --onefile --console matrix.py
   ```
3. Заберите готовый файл `matrix.exe` в появившейся папке **`dist/`**. Теперь его можно запускать с теми же аргументами (например, `matrix.exe -color cyan`) на любом компьютере!
